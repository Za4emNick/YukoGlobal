YUKO GLOBAL — статический сайт каталога кондиционеров

Как запустить:
1. Откройте файл index.html двойным кликом.
2. Для публикации загрузите всё содержимое папки на Vercel, Netlify или обычный хостинг.

Структура:
- index.html — главная страница
- catalog.html — каталог с фильтрами
- brands.html — информация о брендах
- contacts.html — контакты
- products/ — отдельные страницы товаров
- assets/ — стили, JavaScript и локальные изображения
